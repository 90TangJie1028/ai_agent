# 笔记目录

学习笔记、复盘、ADR、面试素材统一放这里，与 `plan/` 计划、`coding/` 代码对应。

## 结构

```text
notes/
  concepts/       # 概念笔记（按学习顺序编号）
  frameworks/     # LangChain / LangGraph / MCP 等
  adr/            # 架构决策记录
  projects/       # 每个练习项目的总结 P1~P6
  daily/          # 每日详细日记（可选，复制 plan/daily/模板.md）
  weekly/         # 周复盘 + 月复盘
  interview/      # 面试题、讲稿、投递记录
```

## 命名规范

| 类型 | 格式 | 示例 |
|:---|:---|:---|
| 概念 | `NN-主题.md` | `01-LLM-API基础.md` |
| 周复盘 | `week-NN-复盘.md` | `week-01-复盘.md` |
| 月复盘 | `month-NN-复盘.md` | `month-01-复盘.md` |
| 项目总结 | `PN-总结.md` | `P1-总结.md` |
| ADR | `NNN-标题.md` | `001-retry-policy.md` |
| 每日 | `YYYY-MM-DD.md` | `2026-06-22.md` |

## 推荐积累（18 周底）

- `concepts/` ≥ 10 篇
- `adr/` ≥ 3 篇
- `projects/P1` ~ `P6` 各 1 篇总结
- `interview/问答集.md` ≥ 30 题
- `weekly/week-01` ~ `week-18` 复盘齐全
