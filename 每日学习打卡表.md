## 使用说明

1. **每天学完后**：在对应日期行打勾，填写「今日产物」和「明日第一件事」。
2. **每天必须留 1 个产物**：commit / 测试结果 / README / 评测表 / 架构图 / demo 截图 任选其一。
3. **每周日**：复制 [周复盘模板](#周复盘模板) 到 `logs/week-XX-复盘.md` 并 commit。
4. **详细日记**：可写在 `logs/daily/YYYY-MM-DD.md`（可选，打卡表只记要点）。

### 每日 4 小时节奏

| 时段 | 时长 | 内容 |
|:---|:---|:---|
| 学习 | 30 min | 官方文档 / 源码 → 5 条笔记 + 1 个可落地动作 |
| 编码 | 120 min | 当日主任务 → 可运行代码 |
| 测试 | 45 min | 测试、评测、调试 |
| 文档 | 30 min | README / ADR / 复盘 |
| 计划 | 15 min | 写下明天第一个动作 |
