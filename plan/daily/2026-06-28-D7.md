## Pydantic v2 30 分钟导读（D7 学习）

W2 D8 会写 `schemas.py`，今天先建立直觉：

| 概念 | 作用 | TS 类比 |
|:---|:---|:---|
| `BaseModel` | 声明字段 + 自动校验/序列化 | `interface` + 运行时 zod |
| `Field(default=..., description=...)` | 元数据与约束 | zod `.describe().min()` |
| `model_validate(dict)` | dict → 对象，失败抛 `ValidationError` | `schema.parse()` |
| `model_dump()` | 对象 → dict | 展开对象 |
| `model_json_schema()` | 生成 JSON Schema | 给 OpenAI `tools` 用 |

**与 Gateway 的关系**：D5 的 `CallRecord` 是 `@dataclass`——够用但不校验。W2 模型输出要 parse 成 `StructuredAnswer`，非法 JSON 要 catch `ValidationError` 并重试。

预习命令（可选）：

```powershell
.\.venv\Scripts\python.exe -c "from pydantic import BaseModel, Field; class A(BaseModel): x: int = Field(ge=0); print(A.model_json_schema())"
```
