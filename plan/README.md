## 目录约定

```text
plan/          ← 你正在看的：计划与打卡
coding/        ← 所有代码与练习项目
notes/         ← 学习笔记、复盘、面试素材
```

**每天流程**：打开当周 `weekly/week-XX.md` → 按 4 时段执行 → 笔记写入 `notes/` → 代码 commit 到 `coding/` → 在 `daily/打卡索引.md` 打勾。
